# Sequential MNIST
## Usage

```
python sMNIST_task.py [args]
```

Options:
- nhid : hidden size of recurrent net
- epochs : max epochs
- batch : batch size
- lr : learning rate
- dt : step size parameter dt of the coRNN
- gamma : y controle parameter gamma of the coRNN
- epsilon : z controle parameter epsilon of the coRNN
